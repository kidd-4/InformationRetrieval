Tweeten via sms




Als je 
jouw mobiele telefoonnummer aan je Twitter-account toevoegt
, kan je Tweets sturen door een sms naar een short code of long code te sturen.


Door een sms naar een van deze short code- of long code-telefoonnummers te sturen, wordt je bericht als Tweet geplaatst in je Twitter-profiel (en verstuurd naar al je volgers).




Er bestaan enkele belangrijke verschillen tussen het gebruik van Twitter via short codes of long codes:




Een short code gebruiken




Als je Twitter gebruikt door sms-berichten te sturen naar een short code, kan je acties uitvoeren en toegang krijgen tot inhoud, net als dit mogelijk is via internet en slimme apparaten.


Als je short codes gebruikt voor je Twitter-account, kan je Tweets plaatsen, meldingen ontvangen en beveiligingsfunctionaliteit (zoals 
inlogverificatie
) activeren.




Momenteel bieden we ondersteuning voor Tweets in twee richtingen (verzenden en ontvangen) via sms-short codes die worden aangeboden door ondersteunde mobiele providers.




Als je nog geen Twitter-account hebt, lees je meer over 
hoe je een nieuwe Twitter-account registreert via sms
.




Als je al een Twitter-account hebt en wil gaan tweeten via sms, ga je als volgt te werk:




Zoek de short code van jouw mobiele provider.


Stuur het woord 
START
 of 
REGISTREER
 naar de short code van jouw mobiele provider.
Gebruik de volgende speciale 
Twitter-opdrachten
 om acties uit te voeren voor je Twitter-account


Een long code gebruiken


Als voor jouw land en provider geen short code wordt weergegeven, kan je mogelijk Tweets plaatsen met behulp van één van de long codes voor Twitter.


De onderstaande long codes werken maar in één richting. Je kan Tweets verzenden die in je profiel worden geplaatst (en naar volgers worden verstuurd), maar je kan via deze methode geen Tweets of andere Twitter-meldingen ontvangen op je telefoon.










Opmerking
: We raden gebruikers van long codes dan ook af om functionaliteiten zoals inlogverificatie te gebruiken, omdat de ervaring mogelijk niet optimaal is.












Abonnees hoeven zich niet in het Verenigd Koninkrijk of Duitsland te bevinden om de onderstaande long codes te gebruiken. Hou er wel rekening mee dat er kosten voor internationale berichtverzending in rekening kunnen worden gebracht wanneer je de codes gebruikt.
